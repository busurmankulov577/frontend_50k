1) Правила подключения стилей прямо на странице 
    используя тег style
    <style>
        Правила стилей
    </style>

    - Данный кусок кода должен стоять перед куском кода <head>

2) Правила написание стилей 
    - Начинается все с того , кому мы должны присвоить стили?
        class
        id
        tag
        attribute
    - мы должеы открыть и закрыть фигирную скобку 
        { }

        Целостная картина равна;
        class {

        }
        id{

        }
        tag{

        }
        attribute{

        }
3) заркывающая фигирная скобка должна быть паралельна названию тега или класса
